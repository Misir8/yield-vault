# DeFi Deposit Platform - Обзор проекта

## Архитектура проекта

Проект состоит из трёх основных компонентов:

### 1. Smart Contracts (Solidity)
- **DepositContract.sol** - основной контракт для управления депозитами
- Функции:
  - `deposit()` - приём депозитов в стабильных токенах (USDT/USDC)
  - `withdraw()` - вывод средств с начисленными процентами
  - `calculateInterest()` - расчёт процентов на основе данных оракла
  - `getBalance()` - проверка баланса пользователя
- Интеграция с Chainlink для получения рыночных ставок

### 2. Oracle Service (Go)
- REST API для взаимодействия с блокчейном
- Мониторинг событий контракта
- Агрегация данных о процентных ставках
- Обновление данных в смарт-контракте

### 3. Frontend (React)
- Интерфейс для пользователей
- Подключение Web3 кошельков (MetaMask)
- Отображение баланса и начисленных процентов
- Формы для депозита и вывода средств

## Технологический стек

### Blockchain
- Solidity ^0.8.20
- Hardhat для разработки и тестирования
- OpenZeppelin для безопасных контрактов
- Chainlink для оракулов

### Backend
- Go 1.21+
- go-ethereum для взаимодействия с блокчейном
- Gin/Echo для REST API

### Frontend
- React 18+
- ethers.js / web3.js
- Wagmi для Web3 интеграции
- TailwindCSS для стилей
